## 代码提交格式

## 代码提交路径

将你的代码按照赛题组格式要求提交到`mindquantum`仓`research`分支的`paper_recurrence`文件夹的`xxx`文件夹内，`xxx`文件夹命名格式如下：

`paperid_username`

其中`paperid`为论文的序号，`username`为你的参赛用户名，例如: `1_donghufeng`.
